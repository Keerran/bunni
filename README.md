# Bunni
Bunni is a desktop app built to read manga.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [Rust](https://www.rust-lang.org/)

### Installation

1. Clone the repository
```bash
git clone https://github.com/keerran/bunni
cd bunni
```

2. Install the dependencies
```bash
npm install
```

3. Run the app
```bash
npm run tauri dev
```
